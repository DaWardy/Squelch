# Squelch Assets

This folder contains runtime data files.

## Contents

- `artemis_signals.json` — Artemis signal database (downloaded on first use)
- `squelch.ico` — Application icon (TODO: create before v1.0)
- `cty.dat` — CTY DXCC country data (downloaded on first use)

## Notes

Files in this folder are not committed to git
(see .gitignore). They are downloaded/generated
at runtime by the application.
